# 🛰 Satellite Tracker Web App

A simple web application to track satellites in real-time using a frontend interface and a Python backend API.

This project demonstrates basic full-stack development using **HTML, CSS, JavaScript, and Python (Flask)**.

# 🚀 How to Run the Project

## Step 1: Clone the Repository

git clone https://github.com/thekarthek/satellite-tracker-x.git

---

## Step 2: Navigate to the Project Folder

cd satellite-tracker-x

---

## Step 3: Run the Backend Server

Navigate to backend folder

cd backend

Install Flask if not installed

pip install flask

Run the server

python app.py

The backend server should start.

---

## Step 4: Open the Frontend

Go to the frontend folder

cd ../frontend

Open the website

start index.html

The satellite tracker interface will open in your browser.

---

# 🧠 How It Works

1. The frontend UI is built using HTML, CSS, and JavaScript.
2. JavaScript sends requests to the Python backend.
3. The backend processes satellite data.
4. The frontend displays the satellite information.
